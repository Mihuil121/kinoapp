# 🎬 КиноВселенная

SPA-приложение для просмотра информации о фильмах на основе [Kinopoisk API](https://kinopoisk.dev/).

## Стек технологий

- **React 18** + **TypeScript**
- **React Router v6** — маршрутизация
- **Axios** — HTTP-запросы
- **Vite** — сборщик

## Функциональность

- 🎥 Список фильмов с бесконечным скроллом (50 фильмов за раз)
- 🔍 Поиск фильмов
- 🎭 Фильтры по жанру, рейтингу, году выпуска (синхронизированы с URL)
- 📄 Детальная страница фильма
- ❤️ Избранное (сохраняется в localStorage)
- ⇄ Режим сравнения двух фильмов

## Запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка API ключа

Получите бесплатный API-ключ на [kinopoisk.dev](https://kinopoisk.dev/).

Создайте файл `.env` в корне проекта:

```bash
cp .env.default .env
```

Откройте `.env` и вставьте ваш ключ:

```
VITE_KINOPOISK_API_KEY=ВАШ_КЛЮЧ_ЗДЕСЬ
```

### 3. Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

### 4. Сборка для продакшена

```bash
npm run build
npm run preview
```

## Структура проекта

```
src/
├── api/          # Запросы к Kinopoisk API
├── components/   # Переиспользуемые компоненты
├── hooks/        # Кастомные хуки (useMovies, useCompare)
├── pages/        # Страницы приложения
├── store/        # Работа с localStorage (избранное)
├── types/        # TypeScript типы
└── index.css     # Глобальные стили
```
